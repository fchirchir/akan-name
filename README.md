# Akan Name Application
#### Akan Name Application, 31st May, 2020
#### By **Felix Chirchir**
## Description
This is a web application that takes your birthday and calculates the day of the week you were born and prints out your Akan Name based on your gender.
## Setup/Installation Requirements
* Download a zip file of the project.
* Extract the files
* open index.html in a browser
* easy-to-understand
